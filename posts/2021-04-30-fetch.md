# Fetch

Envia requisições assíncronas para o servidor. Serve para acessarmos/escrevermos dados em apis externas.

```javascript
fetch('https://ranekapi.origamid.dev/wp-json/api/produto')
  .then((response) => response.json())
  .then((json) => {
    console.log(json);
  });
```
