O método **push()** adiciona um ou mais elementos ao final de um array e retorna o novo comprimento desse array.



```javascript
var numeros = [1, 2, 3];
numeros.push(4);

console.log(numeros); // [1, 2, 3, 4]

numeros.push(5, 6, 7);

console.log(numeros); // [1, 2, 3, 4, 5, 6, 7]
```
